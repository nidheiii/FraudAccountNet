🚀 FraudAccountNet

Transaction-Based Fraud Detection System

📌 Overview

FraudAccountNet is a rule-based system that detects fraudulent banking transactions using pattern analysis and risk scoring. It identifies mule account behavior and supports Neo4j-based graph analysis.

✨ Features
Fraud pattern detection (10 types)
Transaction risk scoring
Account-level risk analysis
Alert prioritization
Neo4j integration

📊 Output

Stored in outputs/:

transactions_with_all_patterns.csv → patterns, risk score, alerts
account_risk_scores.csv → account risk metrics

🎯 Use Cases
Banking fraud detection
Mule account identification
Transaction monitoring
